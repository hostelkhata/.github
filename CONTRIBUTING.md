# Contributing to hostelkhata

Thanks for your interest in hostelkhata.

This guide applies to all repositories under the [github.com/hostelkhata](https://github.com/hostelkhata) organization.

## Current Status

All hostelkhata repositories are private and proprietary. We are not accepting external contributions at this time.

## How to Help

1. **Join the waitlist** at [hostelkhata.in](https://hostelkhata.in)
2. **Report bugs** via GitHub Issues
3. **Share feedback** at [hello@hostelkhata.in](mailto:hello@hostelkhata.in)
4. **Refer a hostel owner** â€” word of mouth is our growth engine

## Future

If hostelkhata succeeds and we build a community, we'll open up contributions for:
- Bug fixes
- Documentation improvements
- Feature suggestions
- Accessibility improvements
- Localization (Hindi, Telugu, Kannada, etc.)

## Code Style

- TypeScript strict mode
- Prettier for formatting
- ESLint for linting
- Tailwind CSS for styling

## Commit Convention

```
type(scope): description

feat(ledger): add bulk charge entry
fix(auth): resolve OTP timeout on slow networks
docs(landing): update pricing section
```

Types: `feat`, `fix`, `docs`, `style`, `refactor`, `test`, `chore`
