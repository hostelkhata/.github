## The Problem

Hostel owners in India still track tenant dues in registers and notebooks. Payment disputes are common. Reminder calls are awkward. Hours are wasted on bookkeeping every month.

## The Solution

**hostelkhata** — simple hostel and PG management software built for India.

| Feature | What it does |
|---------|-------------|
| Tenant Dashboard | See all dues at a glance |
| WhatsApp Reminders | One-tap reminders with pre-filled balance |
| Payment Tracking | Cash, UPI, bank transfer — all in one place |
| Mobile First | Works on any Android phone, no app needed |
